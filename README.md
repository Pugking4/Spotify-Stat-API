# Spotify-Stat-API

Spring Boot API for Spotify listening analytics from Spotify-Stat-Tracker track history. </br>

Spotify-Stat-API queries a PostgreSQL database of your complete Spotify playback history to generate advanced statistics including top tracks, artist distributions, listening heatmaps, session analytics, and niche/popular content discovery. Works with data from the companion Spotify-Stat-Tracker. </br>
